# mainhome-aiyouxi-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库内容不针对任何特定网站或域名，仅作为静态 HTML 文件的存储与展示空间。

## 项目简介

本仓库收集并托管了若干独立的 HTML 页面，这些页面可能包含信息展示、工具界面、小型应用或实验性内容。  
所有文件均为纯前端实现，无需后端服务支持，可直接通过浏览器访问。

## 目录说明

- `/pages/` — 存放各独立 HTML 页面文件
- `/assets/` — 页面所需的公共资源（CSS、JavaScript、图片等）
- `/docs/` — 与页面相关的文档或说明（如有）

每个 HTML 文件通常自包含其样式与逻辑，部分页面可能引用 `/assets/` 下的共享资源。

## 页面归档说明

- 所有页面均为独立归档，互不依赖
- 页面内容可能随时间更新或移除，不保证长期一致性
- 归档仅用于记录与展示，不代表对页面内容的持续支持或背书

## 使用方式

1. 克隆或下载本仓库
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件
3. 也可通过 GitHub Pages 或其他静态托管服务部署后访问
